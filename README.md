# Vulkan Learning Engine

Этот репозиторий — персональная "песочница" для глубокого изучения графического API **Vulkan** и архитектуры 3D-движков на C++.

Проект создается в образовательных целях. Цель этого проекта — пройти путь от инициализации базовых структур API до создания полноценного и расширяемого 3D-рендерера, разобравшись в каждом байте памяти под капотом.

## Технологический стек

* **Язык:** C++20
* **Графический API:** Vulkan
* **Оконная система:** GLFW
* **Математика:** GLM
* **Сборка:** CMake
* **Шейдеры:** GLSL (с автоматической компиляцией в SPIR-V через `glslc` прямо в CMake)

## Как собрать проект

**Требования:**

1. Установленный **Vulkan SDK**.
2. **CMake** (версии 3.15 или выше).
3. Компилятор с поддержкой C++20 (MSVC, GCC или Clang).

**Шаги сборки:**

```bash
# 1. Клонируй репозиторий
git clone https://github.com/ТВОЙ_НИК/VulkanEngine.git
cd VulkanEngine

# 2. Создай папку для сборки и сгенерируй файлы CMake
mkdir build
cd build
cmake ..

# 3. Собери проект
cmake --build .

```
